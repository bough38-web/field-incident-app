# 사고보고 관리자 다운로드 ZIP (고도화 템플릿)

이 ZIP은 관리자 다운로드 기능을 구현할 때 사용할 수 있는 **완성형 출력물 포맷**(폴더/파일 규격) 템플릿입니다. 단순한 기능 구현을 넘어 메모리 최적화, 속도 개선, 그리고 **데이터 시각화 대시보드**까지 포함되도록 고도화되었습니다.

## 🚀 주요 개선 사항 (고도화)
1. **메모리 최적화**: 다운로드 파일 용량이 커져 OOM(Out of Memory) 기반 서버 다운을 방지하기 위해 `io.BytesIO` 대신 일정 용량 이상 시 디스크를 활용하는 `tempfile.SpooledTemporaryFile` 방식을 도입했습니다.
2. **속도 비약적 향상**: `concurrent.futures.ThreadPoolExecutor`를 사용해 S3 등 원격 저장소에서 사진 파일들을 멀티 스레딩으로 **동시 병렬 다운로드** 받도록 수정되었습니다.
3. **오류 극복 (검증 로직)**: 일부 사진 파일이 유실되거나 권한 오류가 발생해도 파일 다운로드가 중간에 멈추지 않고, 오류 처리 후 나머지 파일들을 계속 압축할 수 있도록 안전한 구조가 적용되었습니다. 

## 📊 시각화 (Visualization) 기능
생성되는 `incidents.xlsx` 파일을 열어보면 아래 두 개의 시트가 구성됩니다.
1. **[Summary & Dashboard] 시트**: 전체 사고 건수 요약 및 **[지점별 사고 건수 막대 차트 (BarChart)]**가 자동으로 생성되어 시각화됩니다.
2. **[Incidents List] 시트**: 기존의 개별 사건/사고 상세 리스트가 기록되어 있으며, 엑셀 필터와 헤더 색상이 자동 적용되어 있습니다.

## 📂 폴더 구조 및 ZIP 구성
- `incidents.xlsx` : 시각화 대시보드 + 필터 결과 목록
- `photos/` : 사진이 카테고리(컬럼)별 폴더로 자동 정렬되어 저장
  - `01_외관/`
  - `02_계량기/`
  - `03_현장/`
  - `99_기타/`
- `categories.yaml` : 카테고리 정의/추가 (예: 설정 파일 수정만으로 폴더 제어)
- `export_builder.py` : S3 호출 및 ZIP 압축 로직을 담은 코어 엔진 코드
- `app_example.py` : **(추가됨)** 이 코드를 어떻게 서버로 띄워 서비스하는지 보여주는 FastAPI 웹 서버 예제코드.

## ☁️ 배포 진행방법 안내 (Deployment Guide)

이 모듈을 실제 운영 환경에 배포하여 프론트엔드와 연동하는 방법 3가지를 제안합니다.

### 1. 실시간 스트리밍 방식 (추천: 일반적인 트래픽 수준)
- **구현 방식**: 동봉된 `app_example.py`처럼 파일 압축을 메모리에서 수행하되, 생성되는 즉시 `StreamingResponse`로 클라이언트 브라우저에 청크(chunk) 단위로 밀어주는 방식.
- **배포 방법**: 
  - 코드를 Docker 이미지로 패키징하여 **AWS ECS(Fargate)** 나 일반 서버 인스턴스에 배포합니다.
  - 명령어: `uvicorn app_example:app --host 0.0.0.1 --port 8000 --workers 4`
- **장점**: 대기 시간 없이 브라우저에서 다운로드가 바로 시작됩니다.

### 2. 비동기 백그라운드 워커 방식 (추천: 사고 사진이 500장~수만장일 때)
- **구현 방식**: 다운로드 요청(API)이 들어오면 서버는 "다운로드가 예약되었습니다" 라는 응답만 주고, 백그라운드 Worker(**Celery** + Redis, 또는 AWS SQS 연동)가 `build_export_zip` 프로세스를 실행. 완료 시 S3 버킷에 완성된 `.zip`을 업로드하고 관리자 이메일로 "다운로드 링크"를 전송합니다.
- **장점**: 수 만장의 사진다운로드 작업이 웹서버의 생명주기(Timeout 30초 등) 에 의해 끊길 위험이 없습니다.

### 3. Serverless 방식 (AWS Lambda)
- **제약 사항 주의**: AWS Lambda의 경우 `/tmp` 디스크 용량이 512MB~10GB이고, 실행 한도 시간이 최대 15분입니다. 
- **장점**: 관리할 서버가 없이 트래픽에 맞춰 무한 확장이 가능합니다.
- **구현 방식**: 다운로드 요청이 발생하면 Lambda가 실행되어 비동기 방식으로 압축 후 결과를 다운가능한 presigned-URL로 반환합니다.

## 실행해보기 (로컬 테스트용)
### 1. 다운로드 API 서버 실행
```bash
pip install -r requirements.txt
python app_example.py
```
브라우저에서 `http://127.0.0.1:8000/api/export/incidents` 에 접속하면 목업 사진이 들어있는 ZIP 파일이 성공적으로 다운로드 됩니다.

### 2. 로컬 통계 대시보드 (Streamlit) 실행
```bash
pip install streamlit pandas plotly
streamlit run dashboard_app.py
```
실행 시 자동으로 브라우저가 열리며 `http://localhost:8501` 에서 인터랙티브 대시보드를 시각적으로 확인할 수 있습니다.
